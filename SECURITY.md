# Security Policy

## Reporting a Vulnerability

We take the security of our systems seriously. If you have found a security vulnerability, please report it to us immediately.

Please report vulnerability details by emailing **tsainez@gmail.com**.

We will acknowledge your report within 48 hours and provide an estimated timeframe for a fix.

## Security Measures

This site is a static site generated by Jekyll and deployed on Netlify. We implement the following security measures:

- **Strict Content Security Policy (CSP)**: We restrict the sources of scripts, styles, and other resources to trusted domains.
- **HTTP Security Headers**: We enforce HSTS, X-Frame-Options, X-Content-Type-Options, and Referrer-Policy.
- **Minimal Client-Side Logic**: We minimize the use of client-side JavaScript to reduce the attack surface.

## Supported Versions

As a continuously deployed static website, the latest version is the only supported version. Please ensure you are viewing the live site.
